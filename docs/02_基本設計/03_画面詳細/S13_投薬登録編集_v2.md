# S13 投薬登録/編集 画面詳細（v2）

## 1. 画面概要
- **目的**: 投薬（薬名・期間・頻度・メモ）を分かりやすく記録する。
- **URL**:
  - 新規: `/pets/:petId/medications/new`
  - 編集: `/pets/:petId/medications/:medId/edit`

---

## 2. 画面項目一覧

| No | 物理名 | 項目名 | 種類 | 必須 | 入力仕様 | 備考 |
|---:|---|---|---|---|---|---|
| 1 | name | 薬名 | テキスト | 必須 | 最大100 | |
| 2 | dosage | 用量 | テキスト | 任意 | 最大100 | 例: 1/2錠 |
| 3 | frequency | 頻度 | テキスト | 任意 | 最大100 | 例: 1日2回 |
| 4 | start_on | 開始日 | 日付 | 推奨 | 初期=今日 | |
| 5 | end_on | 終了日 | 日付 | 任意 | 未設定可 | 継続中扱い |
| 6 | note | メモ | テキストエリア | 任意 | 最大1000 | |
| 7 | actions.save | 保存 | ボタン | - | | |
| 8 | actions.cancel | キャンセル | ボタン | - | 一覧へ | |
| 9 | actions.delete | 削除 | ボタン | 任意 | 編集時のみ | |

---

## 3. バリデーション
- name必須
- start_on/end_on日付形式
- end_onはstart_on以降（任意）

---

## 4. 操作フロー
- 保存 → S12へ戻る
