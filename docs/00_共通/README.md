# 00_共通 フォルダの使い方

Pet Medical Record App（ペットカルテアプリ）

---

## 📂 このフォルダの目的

プロジェクト全体で **共通して参照すべきルール・定義** をまとめています。

AI・開発者が実装時に **必ず読む** べきドキュメント群です。

---

## 📄 ファイル構成

| ファイル | 内容 | 重要度 |
| --- | --- | --- |
| `00_デザイン.md` | UI/UX・トンマナ定義 | ★★★ |
| `01_用語定義.md` | ドメイン用語・表記統一 | ★★★ |
| `02_命名規則.md` | ファイル・変数・API命名 | ★★★ |
| `03_実装方針.md` | 技術選定・コーディングルール | ★★★ |
| `04_禁止事項.md` | AI暴走防止ルール（超重要） | ★★★★★ |
| `README.md` | このファイル | - |

---

## 🛠️ 使い方

### 新規実装時

1. `00_デザイン.md` でUI方針を確認
2. `01_用語定義.md` で用語を統一
3. `02_命名規則.md` で命名方法を確認
4. `03_実装方針.md` で技術スタック・構成を確認
5. `04_禁止事項.md` でNG項目をチェック

### AIに依頼する際

以下をプロンプトに含める：

```
以下のドキュメントを遵守して実装してください。
- docs/00_共通/00_デザイン.md
- docs/00_共通/02_命名規則.md
- docs/00_共通/03_実装方針.md
- docs/00_共通/04_禁止事項.md
```

---

## ⚠️ 注意事項

- このフォルダのルールは **プロジェクト全体の最優先ルール**
- 矛盾がある場合は、このフォルダのルールを優先
- 変更時は必ずチーム（or 自分自身）で合意
