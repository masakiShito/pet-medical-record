# 命名規則

Pet Medical Record App（ペットカルテアプリ）

---

## 1. 目的

本書は、**コード・ファイル・API・DB等の命名を統一し、可読性を保つ**ことを目的とする。

---

## 2. 基本方針

- 英語表記を基本とする
- 略語は最小限にする
- 意味が明確に伝わる名前を選ぶ

---

## 3. ファイル・ディレクトリ命名

### 3.1 フロントエンド（React）

| 種類 | 形式 | 例 |
| --- | --- | --- |
| コンポーネント | PascalCase | `PetCard.jsx` |
| hooks | camelCase + use prefix | `usePets.js` |
| utils | camelCase | `dateFormatter.js` |
| pages | PascalCase | `PetListPage.jsx` |

### 3.2 バックエンド（Python / FastAPI）

| 種類 | 形式 | 例 |
| --- | --- | --- |
| モジュール | snake_case | `pet_service.py` |
| クラス | PascalCase | `PetRepository` |
| 関数 | snake_case | `get_pet_by_id()` |

---

## 4. 変数・関数命名

### 4.1 JavaScript / React

| 種類 | 形式 | 例 |
| --- | --- | --- |
| 変数 | camelCase | `petList`, `isLoading` |
| 定数 | UPPER_SNAKE_CASE | `API_URL`, `MAX_PETS` |
| 関数 | camelCase | `fetchPets()`, `handleSubmit()` |
| イベントハンドラ | handle + 動詞 | `handleClick`, `handleChange` |

### 4.2 Python

| 種類 | 形式 | 例 |
| --- | --- | --- |
| 変数 | snake_case | `pet_list`, `is_deleted` |
| 定数 | UPPER_SNAKE_CASE | `DATABASE_URL` |
| 関数 | snake_case | `get_all_pets()` |
| クラス | PascalCase | `PetService` |

---

## 5. API命名

### 5.1 エンドポイント

- RESTful設計に従う
- 複数形リソース名を使用
- 動詞は使わない（HTTPメソッドで表現）

| メソッド | エンドポイント | 説明 |
| --- | --- | --- |
| GET | `/pets` | ペット一覧取得 |
| POST | `/pets` | ペット新規登録 |
| GET | `/pets/{id}` | ペット詳細取得 |
| PUT | `/pets/{id}` | ペット更新 |
| DELETE | `/pets/{id}` | ペット削除 |
| GET | `/pets/{id}/records` | 記録一覧取得 |

### 5.2 レスポンスキー

- camelCase（JSON）
- snake_case（Python内部）

---

## 6. データベース命名

### 6.1 テーブル・カラム

| 種類 | 形式 | 例 |
| --- | --- | --- |
| テーブル | snake_case + 複数形 | `pets`, `records` |
| カラム | snake_case | `pet_id`, `recorded_on` |
| 主キー | id | `id` |
| 外部キー | 参照先_id | `pet_id`, `user_id` |

### 6.2 インデックス

- `idx_{table}_{column}`
- 例: `idx_pets_user_id`

---

## 7. 環境変数命名

- UPPER_SNAKE_CASE
- プレフィックスで分類

| プレフィックス | 例 |
| --- | --- |
| DATABASE_ | `DATABASE_URL` |
| API_ | `API_PORT` |
| CORS_ | `CORS_ORIGINS` |

---

## 8. 禁止事項

- 1文字変数（ループ変数以外）
- 日本語ローマ字変数（`namae`, `touroku` など）
- 曖昧な略語（`tmp`, `data`, `info` のみ）
